Sobre os dados

No exerccio de hoje, vamos utilizar as variveis:

  • fare: indica o valor pago pela passagem, em dlares ($).
  • survived: indica se o passageiro sobreviveu ao naufrgio, sendo 1 para sobreviveu e 0 para no sobreviveu.
  • embark_town: indica a cidade onde o passageiro embarcou no navio.
  • sex: indica o sexo do passageiro, sendo male para homens e female para mulheres.
