Contudo, vamos ver como podemos ver a lista de categorias de uma varivel categrica, para isso vamos usar o mtodo unique do pandas.

Agora, podemos fazer uma ltima dupla de ajustes interessantes.

Primeiro, vamos adicionar um detalhe apenas esttico marcando o fim de cada barra de erro. Para isso vamos adicionar o argumento capsize=0.2. O valor numrico desse argumento serve para controlarmos o tamanho desse trao que marca o limite da barra de erro.

Ainda, vamos aprender a fazer um tipo de ajuste especfico na barra de erro. Podemos us-la de forma a exibir algo semelhante ao que vimos com o boxplot. Para isso vamos criar uma lista no seguinte formato: ["pi", X], onde X um nmero entre 0 e 100 e o percentual dos dados centrais que queremos que a barra de erro cubra. Para um comportamento semelhante ao barplot, podemos usar ["pi", 50]. Assim, a barra de erro cobrir 50% dos dados centrais mas ter a mdia como centro, em vez da mediana (lembrando que por padro o estimador usado a mdia).

Estamos usando a funo da barra de erro referente ao percentual. Para mais detalhes sobre a barra de erros, voc pode consultar a documentao relevante do seaborn. Vamos tratar tambm desse assunto no curso de Cincia de Dados.

In[10]:
sns.pointplot(
    x="idh",
    y="regiao",
    linestyle="",
    capsize=0.2,
    errorbar=["pi", 50],
    data=idh_2022
)
Out[10]:
<Axes: xlabel='idh', ylabel='regiao'>

Poderamos alterar o estimador para a mediana adicionando o argumento estimator="median", contudo vamos manter a mdia.

Vamos adicionar uma ltima alterao alterando o grfico para que ele exiba TODA a faixa das observaes. Para isso, vamos mudar a parte numrica da barra de erro de 50 para 100.

sns.pointplot(

Vimos agora outra forma de observar a distribuio de dados quando enfrentamos um problema que envolve uma varivel categrica e uma numrica: primeiro com a funo boxplot e agora com o pointplot. O pointplot traz menos mincias que o grfico de caixas, mas pode ser uma alternativa mais simples de visualizao.

Nos dois casos poderamos introduzir outra varivel categrica para fazer comparaes mais intricadas, usando o argumento hue.

Uma limitao que observamos tanto no Boxplot quanto no Lineplot que no recebemos nenhuma informao a respeito da quantidade de observaes em cada grupo. Isso pode ser importante a medida que realizamos diferentes cortes. No queremos fazer comparaes entre grupos com milhares e dezenas de observaes sem ter conscincia disso.

Claro, poderamos fazer um histograma, countplot, ou fazer isso apenas pelo cdigo e gerar uma tabela. Outra ideia interessante, que seria tpico das ltimas aulas do curso, seria adicionar um texto em cada caixa com uma informao do nmero de casos em cada grupo.

Contudo, em certos casos podemos usar um grfico para isso ou at juntar dois grficos para ter uma informao visual mais completa. Vamos explorar 2 grficos com esse propsito: o stripplot e o swarmplot.

stripplot e swarmplot: Visualizando cada observao

Tanto o stripplot quanto o swarmplot so grficos que permitem observar a distribuio de uma varivel numrica dentro de grupos diferentes. A ideia que iremos criar um ponto para cada observao (uma linha, se nossos dados estiverem devidamente tidy): em um eixo a posio indica seu valor numrico e em outro ele fica agrupado pela varivel categrica, assim como em grficos de barra.

Vamos comear fazendo um stripplot com o IDH de 2022, agrupado por regio.

In[14]:
sns.stripplot(
    x="idh",
    y="regiao",
    data=idh_2022
)
Out[14]:
<Axes: xlabel='idh', ylabel='regiao'>

O swarmplot tem o mesmo propsito do stripplot, e suas opes so quase inteiramente iguais. A diferena que o swarmplot impede que os pontos se sobreponham. Isso pode ser til em certas situaes, mas se temos muitos pontos pode ser um problema tanto visualmente, quanto em termos de tempo de processamento (nosso grfico pode demorar para ser gerado).

Veja abaixo o mesmo grfico, apenas alterando a funo de plotagem.

In[15]:
sns.swarmplot(
    x="idh",
    y="regiao",
    data=idh_2022
)
Out[15]:
<Axes: xlabel='idh', ylabel='regiao'>

Uma informao importante de ter em mente que a localizao exata no eixo com a varivel categria no importante. H sim uma diferena qualitativa entre os grupos diferentes, e devemos evitar que haja confuso entre os pontos ao fazer o grfico.

Grfico Exploratrio

Nessa aula no vamos fazer um grfico necessariamente mais apresentvel para um visualizao concluda. Mencionamos acima que poderamos combinar dois grficos para ter uma visualizao mais completa: facilitar a visualizao da distribuio e ter uma ideia do nmero de observaes.

Vamos fazer um grfico assim.

Para isso, combinamos o stripplot com o boxplot. Voc no precisa se preocupar inteiramente com como fazer com esse grfico agora, mas perceba que o mecanismo principal no mesmo eixo (ax) colocar os dois grficos sobrepostos, como se desenhassemos uma figura sobre a outra.

Mas isso apenas uma ideia de como usar as ferramentas do curso. Com exceo das 2 ltimas linhas, todo o cdigo abaixo j em grande parte conhecido por ns.

Precisamos usar tambm alguns argumentos diferentes, dodge (este s necessrio em verses mais antigas do seaborn, como a presente no laboratrio) e alpha (controla a transparncias das cores) no stripplot.

Nas ltimas verses (a partir da 0.13) esse cdigo poderia ser ligeiramente mais simples nos dando mais controle, a partir de alguns argumentos diferentes. Mas importante ver que muitas vezes podemos contornar as prprias limitaes do pacote utilizando os recursos da funo, por isso importante consultar sua documentao quando queremos fazer algo diferente do padro!

In[18]:
fig, ax = plt.subplots(figsize=(10, 6))

sns.stripplot(
    x="idh",
    y="regiao",
    data=idh_2022,
    hue="regiao", 
    palette="deep",
    dodge=False, 
    alpha=0.5,
    legend=False, # no cria legenda
    ax=ax,
)

sns.boxplot(
    x="idh",
    y="regiao",
    color="white",
    data=idh_2022,
    ax=ax,
)


ax.set_title("Distribuio de IDH por regio")
ax.set_ylabel("")
ax.set_xlabel("IDH em 2022")

# Essa  a nica parte do cdigo que no vimos ainda
# Vamos adicionar linhas para facilitar a leitura dos valores
ax.grid(axis="x", which="major", linestyle="--", linewidth=0.5, color="gray")
# Remove as bordas do grfico, exceto a inferior
sns.despine(ax=ax, left=True)

Concluso

Neste captulo apresentamos como usar o pointplot de algumas formas. O pointplot extremamente verstil, e permite que representemos a evoluo de uma varivel numrica no tempo, mostremos barras de erro ou variao de valores. Seu bom uso depende da compreenso apropriada do problema sendo abordado.

Ainda, vimos tambm como usar o stripplot e swarmplot para visualizar a distribuio de uma varivel numrica em grupos diferentes com nfase em exibir cada uma de nossas observaes. Isso pode ser interessante se queremos passar uma ideia (no geral, no exata) do nmero de observaes em cada grupo.
